    n8n

    О платформе

    n8n (читается как "n-eight-n") — это инструмент для автоматизации, который помогает связывать приложения с API друг с другом и обрабатывать данные практически без программирования или с минимальным его использованием.

    Ключевые особенности

    Гибкость настройки : Создавайте рабочие процессы любой сложности и разрабатывайте собственные ноды под ваши задачи.

    Удобство развертывания : Запускайте n8n через npm, Docker или используйте облачный хостинг, если не хотите заниматься инфраструктурой самостоятельно.

    Приватность и безопасность : Разворачивайте n8n на собственных серверах для полного контроля над данными и соблюдения требований безопасности.

    Установка в Docker

    Docker — это самый простой и быстрый способ начать работу с n8n. Контейнеризация обеспечивает изолированную среду и упрощает управление зависимостями.

    Быстрый запуск

    Запустите n8n одной командой:

    docker run -it --rm \
  --name n8n \
  -p 5678:5678 \
  docker.n8n.io/n8nio/n8n

    После запуска откройте браузер и перейдите по адресу: http://localhost:5678

    Совет

    Эта команда запускает n8n в режиме тестирования. После остановки контейнера все данные будут удалены. Для сохранения данных используйте volumes.

    Запуск с сохранением данных

    Для продакшн-использования монтируйте директорию для хранения данных:

    docker run -it --rm \
  --name n8n \
  -p 5678:5678 \
  -v ~/.n8n:/home/node/.n8n \
  docker.n8n.io/n8nio/n8n

    Обновление n8n

    Чтобы обновить n8n до последней версии:

    1. Остановите контейнер:
    docker stop n8n
    1. Удалите старый контейнер:
    docker rm n8n
    1. Загрузите новый образ:
    docker pull docker.n8n.io/n8nio/n8n
    1. Запустите контейнер снова с той же командой, что и раньше.

    Важно

    Перед обновлением рекомендуется создать резервную копию данных из директории ~/.n8n

    Docker Compose

    Для более удобного управления используйте docker-compose.yml:

    version: '3.8'

services:
  n8n:
    image: docker.n8n.io/n8nio/n8n
    container_name: n8n
    restart: unless-stopped
    ports:
      - "5678:5678"
    volumes:
      - ~/.n8n:/home/node/.n8n
    environment:
      - N8N_BASIC_AUTH_ACTIVE=true
      - N8N_BASIC_AUTH_USER=admin
      - N8N_BASIC_AUTH_PASSWORD=your_password

    Запуск:

    docker-compose up -d
